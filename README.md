# Big Data Analysis using PySpark

## Objective
Analyze a dataset using PySpark to demonstrate scalable data processing and derive meaningful insights.

## Tools Used
- Python
- PySpark
- Google Colab
- GitHub

## Dataset
Tips Dataset (Restaurant Billing Data)

## Analysis Performed
- Loaded dataset using PySpark
- Generated summary statistics
- Calculated average bill by day
- Analyzed average tips by gender
- Identified highest billing records

## Outcome
Successfully performed data analysis using Apache Spark and generated insights from the dataset.
